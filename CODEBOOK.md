The original data was modified by:
-imported the training and test data into R
-combined the 2 sets and the subject and activity data
-created a total data frame in R that split all the data out into separate columns
-extracted the mean and std columns
-re-labeled the activity column, changed 1-6 to activity labels shown in activity_labels.txt
-Removed the "." from the 561 variable labels from features.txt.
-outputted an independent tidy data set with the average for each activity and each subject

Subject_ID - ID number for the participants/subjects (1-30)
Activity - 6 activities: WALKING, SITTING, LAYING, WALKING_UPSTAIRS, WALKING_DOWNSTAIRS, STANDING
